z-index

z-index controls stacking order when elements overlap. Higher values appear on top. Only affects positioned elements (position other than static). Default is 0.

Stacked layers

z-index: 0 (back)
z-index: 1 (middle)
z-index: 2 (front)

Modal (fixed overlay + dialog)

Modals use high z-index so they appear above page content. Click to close.
