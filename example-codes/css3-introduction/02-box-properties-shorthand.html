Box Properties & Shorthand

Margin, padding, and border can use shorthand. Order is clockwise: top → right → bottom → left (like a clock).

Example 1: Four values

margin: 5px 10px 15px 20px; → top, right, bottom, left

Example 2: Two values

margin: 10px 5px; → top/bottom 10px, left/right 5px

Example 3: One value

margin: 12px; → all four sides 12px

Example 4: Padding & border shorthand

padding: 8px 16px; and border-width: 1px 2px 3px 4px; (same clockwise order).
