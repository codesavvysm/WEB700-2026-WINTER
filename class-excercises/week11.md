# Week 11 — In-Class Exercise: Relational Database (Postgres)

Use **Neon** and the course example app to work with a relational Postgres database, run it locally, and deploy it.

---

## Requirements

### 1. Neon project

- Register on the **Neon** website and create a **project**.

### 2. Run locally

- Clone or open the example code and run the **web server** on your machine.

Repository: [Relational-Database-Postgres](https://github.com/WPFoundations-Examples/Relational-Database-Postgres)

### 3. Deploy to Vercel

- **Deploy** the project to **Vercel** and submit your **Vercel URL**.
